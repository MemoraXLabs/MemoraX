# Support Guidelines

This repository is part of the MemoraX enterprise acquisition package.
Because it is not an open-source project, traditional community support
channels do not apply.

## How to Get Support
For all inquiries related to:

- acquisition  
- technical questions  
- brand assets  
- verification files  
- documentation  
- transfer process  

contact:

**Email:** hello@memorax.ai  
**Website:** https://memorax.ai  

Support is provided exclusively for potential buyers or evaluators of
the MemoraX brand and identity system.

## Scope of Support
Support covers questions about:

- the contents of this repository  
- included assets  
- deployment structure  
- metadata and verification  
- brand documentation  

It does **not** cover custom development, implementation,
modification, or external integrations unless part of a negotiated
acquisition agreement.
